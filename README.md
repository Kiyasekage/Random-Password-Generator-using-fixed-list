# Random Password Generator

A simple Python program that generates a secure and customizable password. The program allows users to specify the number of letters, numbers, and symbols they want in their password. It then randomly selects characters from each category and optionally shuffles them to create a more unpredictable password. This project is ideal for beginners who want to practice lists, loops, string manipulation, and Python's `random` module.

## Features

* Accepts user input for the number of letters, numbers, and symbols.
* Randomly generates characters from each selected category.
* Optionally shuffles the password for greater randomness.
* Combines all selected characters into a single password.
* Demonstrates the use of lists, loops, strings, and the `random` module.

## How to Run

1. Make sure Python 3 is installed.
2. Run the script:

   ```bash
   python password_generator.py
   ```
3. Enter the desired number of letters, numbers, and symbols.
4. Choose whether to randomize the character order (`Y` or `N`).
5. View the generated password.
